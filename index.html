Configuración Supabase

Pasos de configuración

PASO 1. Crea tu cuenta gratuita en supabase.com y crea un nuevo proyecto.
PASO 2. En el dashboard de Supabase → SQL Editor → pega y ejecuta el archivo supabase_setup.sql que descargaste.
PASO 3. Ve a Project Settings → API. Copia la Project URL y la anon key (pública). Pégalas a la derecha.
PASO 4. Ve a Authentication → Users → Add user → Create new user. Crea tu cuenta de administrador con tu correo Gmail y una contraseña segura.
LISTO. Guarda las credenciales a la derecha. Después podrás iniciar sesión como administrador.
